# Aplikasi Kasir Scan

Aplikasi kasir modern dengan fitur scan barcode, manajemen hutang, dan chatbot AI.

## Struktur Proyek

- **backend/** - Backend API dengan Node.js dan Express
- **frontend/** - Frontend mobile dengan React Native dan Expo
- **kasirchatbot/** - Sistem chatbot dengan AI untuk kasir

## Fitur Utama

- 📱 Scan barcode produk
- 💰 Manajemen transaksi
- 📊 Laporan penjualan
- 👥 Manajemen pelanggan dan hutang
- 🤖 Chatbot AI untuk bantuan
- 📦 Manajemen stok dan produk
- 👨‍💼 Manajemen karyawan

## Instalasi

### Backend
```bash
cd backend
npm install
npm start
```

### Frontend
```bash
cd frontend
npm install
npm start
```

### Chatbot
Lihat dokumentasi di folder `kasirchatbot/` untuk setup lengkap.

## Teknologi

- React Native + Expo
- Node.js + Express
- SQLite
- Google Gemini AI
- TypeScript
